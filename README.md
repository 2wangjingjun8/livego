# livego
go 实现直播服务
### install
    chmod 755 server.sh<br/>
    ./server.sh &<br/>
    (有些情况需要用vim 打开 set ff=unix 然后:wq)
### use
    采用vue+webpack实现ui
    所有在config里
    日志在logs里
    http://localhost:8080/live(视频直播)
    http://localhost:8080/camera(录视频)
